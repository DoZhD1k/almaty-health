# Интеграция GeoJSON данных

## Обзор

Интегрированы географические данные из папки `geo-files` в интерактивную карту здравоохранения.

## Что было добавлено

### 1. GeoJSON слои

- **Административные районы** (`districts.geojson`) - границы районов города
- **Зеленые зоны 10 мин** (`10min_green.geojson`) - парки в 10-минутной доступности
- **Доступность 15 мин** (`15min.geojson`) - зоны 15-минутной доступности до медучреждений
- **Доступность 30 мин** (`30min.geojson`) - зоны 30-минутной доступности
- **Медицинские учреждения** (`Extra_MO_coord.geojson`) - дополнительные медучреждения из GeoJSON
- **Сетка населения** (`pop_grids.geojson`) - плотность населения по квадратам

### 2. Новые компоненты

- `EnhancedFacilityMap.tsx` - расширенная карта с поддержкой слоев
- `LayerControlPanel.tsx` - панель управления слоями карты
- `lib/utils/geojson.ts` - утилиты для работы с GeoJSON данными

### 3. Улучшения

- Мягкие стили слоев с низкой непрозрачностью
- Группировка слоев по категориям (Здравоохранение, Инфраструктура, Доступность)
- Все слои отключены по умолчанию для избежания визуального хаоса
- Улучшенные маркеры медучреждений с белой обводкой
- Увеличенная высота карты (600px)

## Использование

1. Откройте главную страницу приложения
2. Используйте правую панель "Слои карты" для включения нужных слоев
3. Каждый слой можно включить/выключить индивидуально
4. Доступны быстрые действия "Показать все" / "Скрыть все"

## Цветовая схема

- **Районы**: Серый (очень прозрачный)
- **Зеленые зоны**: Светло-зеленый
- **Доступность 15 мин**: Светло-оранжевый
- **Доступность 30 мин**: Светло-красный
- **Медучреждения (GeoJSON)**: Красный
- **Сетка населения**: Светло-синий

## Технические детали

- Файлы расположены в `public/geo-files/`
- Загрузка слоев происходит асинхронно только при включении
- Используется react-leaflet для отображения
- Стили оптимизированы для читаемости карты
